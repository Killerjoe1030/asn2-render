ABOUT THE GRAPH

This state of the art student tracker enables the user to add, edit, delete, and display the students as rectangles on the display page. Each rectangle provides a visual representation of all the attributes of the individual students. For example, the weight of the student determines the overall width of the rectangle while the height of the student determines the height of the rectangle. The name of the student and their GPA are displayed inside the rectangle for the user's convenience.
